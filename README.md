# Vardhman Store Performance Dashboard

## Overview
The **Vardhman Store Performance Dashboard** provides an analytical view of store operations, helping to monitor sales trends, customer behavior, and overall performance.  
Built entirely in **Microsoft Excel**, this interactive dashboard simplifies complex datasets into clear, actionable insights for better business decision-making.

---

## Objective
The goal of this dashboard is to:
- Track store sales and revenue performance.  
- Identify key growth areas and underperforming segments.  
- Improve operational efficiency using data-driven insights.  

---

## Key Insights
- Overall store performance summarized through dynamic visuals.  
- Monthly and yearly sales comparison to understand business growth.  
- Product- and category-level analysis for better inventory management.  
- KPIs such as revenue, profit margin, and sales distribution across stores.  

---

## Tools Used
- **Microsoft Excel**
  - Pivot Tables & Charts
  - Conditional Formatting  
  - Data Validation  
  - Dynamic Dashboards  

---

## Dataset Information
The dataset contains information related to:
- Sales transactions  
- Product categories  
- Store locations  
- Revenue and profit data  

---
## How to Use
1. Download the file **`Vardhman Store Data Report.xlsx`**.  
2. Open it in **Microsoft Excel**.  
3. Use the filters and slicers in the dashboard sheet to explore insights interactively.  

---

## About the Project
This dashboard is a part of a data visualization project aimed at enhancing business performance monitoring using Excel. It demonstrates how retail data can be transformed into clear, visual insights with minimal technical complexity.
